# Robin Cookie Checker Pro

Dashboard web untuk mengecek cookies Roblox dengan mudah.

## Fitur
- ✅ Dashboard web dengan live updates
- ✅ Check multiple cookies sekaligus
- ✅ Live progress tracking
- ✅ Discord webhook notifications
- ✅ Export cookies valid
- ✅ Responsive design (mobile & desktop)

## Cara Deploy
1. Upload semua file ke GitHub
2. Import ke Vercel
3. Buka URL yang diberikan Vercel
4. Login dengan password default: `admin123`

## Cara Ganti Password
Edit file `public/script.js` line 3:
